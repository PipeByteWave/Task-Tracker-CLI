# Task CLI

**Task CLI** es una herramienta de línea de comandos diseñada para gestionar tareas de manera sencilla. Permite crear, listar, actualizar, eliminar y marcar tareas como realizadas desde la terminal.

---

## Funciones Principales

- Crear tareas con título y descripción.
- Listar todas las tareas creadas.
- Actualizar o eliminar tareas existentes.
- Marcar tareas como realizadas.

Para más detalles sobre los comandos disponibles, consulta la sección de [Comandos](comands.md).
